Getting Started

Quick-Start Guide

Get from zero to your first SQL query in under two minutes. MindSQL runs entirely from your terminal — no browser, no cloud, no setup wizard.

1

Download the binary for your OS

Head to the Installation page and grab the build for Windows, macOS, or Linux.

2

Make it executable (macOS / Linux)

Open your terminal, cd to your Downloads folder, then run:

bash 
$ chmod +x MindSQL-macos-latest
$ ./MindSQL-macos-latest

On Windows, just double-click the .exe. If Defender blocks it, click More Info → Run Anyway.

3

Connect to your database

MindSQL will prompt you for a connection string on first launch. Example for MySQL:

connection string 
mysql://user:password@localhost:3306/mydb
4

Ask a question in plain English

Type any natural-language question and MindSQL will generate and run the SQL:

mindsql session 
mindsql> show me all users who signed up last week

-- Generated SQL:
SELECT * FROM users
WHERE created_at BETWEEN
  NOW() - INTERVAL '7 days' AND NOW();

-- 42 rows returned in 12ms

Your First Query

A few example queries to get a feel for what MindSQL can do.

Examples
mindsql> how many orders were placed today?
mindsql> list the top 5 customers by total spend
mindsql> find all products with less than 10 items in stock
mindsql> show average session duration grouped by country
mindsql> delete all users who haven't logged in for 90 days
⚠ Destructive queries (DELETE, DROP, TRUNCATE) are automatically flagged and require a confirmation prompt before execution.

Terminal Commands

Commands available in the interactive MindSQL terminal on the home page. Press Tab to autocomplete, ↑/↓ for history.

Command Description Type
help List all available commands with descriptions. info
about Display information about MindSQL — its purpose, team, and origin. info
version Show version number, build date, NLP engine, and database support. info
team List all core team members and their roles. info
features Print a summary of MindSQL's key capabilities. info
whoami Returns a random motivational message. Because why not. fun
status Run a live diagnostic — checks AI engine, SQL parser, API key, and DB adapters. system
github Open the MindSQL Hugging Face model page in a new tab. nav
clear Clear the terminal output. util

Install Commands

Type these directly into the home terminal to trigger a simulated install and download your OS installer stub.

Command Platform Result
windows, sql --install Windows 10/11 Runs boot sequence, shows Windows ASCII logo, downloads installer stub.
macos, sql --install macOS 12+ Runs boot sequence, shows Apple ASCII logo, downloads installer stub.
ubuntu, sql --install Ubuntu / Debian Runs boot sequence, shows Ubuntu ASCII logo, downloads installer stub.
linux, sql --install Linux (generic) Alias for ubuntu variant.
✓ The terminal also supports Tab autocomplete — start typing a command and press Tab to complete it.

The AI Model

MindSQL uses a custom fine-tuned Qwen2.5-3B model, quantized to GGUF format for fast local inference. No cloud calls, no API key required by default.

FAQ

Common questions answered.

Does MindSQL send my data to the cloud?

No. MindSQL runs its AI model entirely on your machine. Your queries and database contents never leave your device.

Can I use my own API key (e.g. OpenAI)?

Yes. Run setup in the terminal to configure an external API key, which unlocks GPT-4o for higher accuracy translations.

Is MindSQL free?

Yes, MindSQL is fully free and open-source under the MIT license. See the Hugging Face model page for details.

My antivirus flagged the binary. Is it safe?

Yes. Because the binary is unsigned (we're a small indie team), some antivirus tools flag it as unknown. You can click Allow safely. The model weights are publicly available on Hugging Face.
