🚀 Installation & Download Guide

Get MindSQL up and running on your system in minutes.

1. Windows (Recommended)

The easiest way to use MindSQL on a PC.

⊞ Download for Windows

Installation:

  1. Download the .exe file.
  2. Double-click the file to launch.
  3. Note: If Windows Defender shows a "Windows protected your PC" message, click More Info and then Run Anyway.

Logo: Look for the small2big icon on your desktop!

2. macOS (Apple Silicon & Intel)

High-performance AI for MacBooks.

 Download for macOS

Installation:

  • # Open your Terminal app
  • # Move to your downloads folder:
  • $ cd ~/Downloads
  • # Grant permission to run:
  • $ chmod +x MindSQL-macos-latest

First Launch: Right-click the file in Finder and select Open. When the security prompt appears, click Open again.

Pro Tip: This version is optimized with Apple Metal for faster SQL generation.

3. Linux (Ubuntu/Debian)

Fast, terminal-based SQL generation.

🐧 Download for Ubuntu

Installation:

  • # Open your Terminal
  • # Make the file executable:
  • $ chmod +x MindSQL-ubuntu-latest
  • # Run the application:
  • $ ./MindSQL-ubuntu-latest

Dependency Note: If the app doesn't start, ensure you have standard C++ libraries:
sudo apt install libstdc++6

🛠 Troubleshooting & Support

MindSQL doesn't open?

Ensure you have downloaded the correct version for your Operating System.

Anti-Virus Blocks

Because this is an unsigned binary from our small2big development team, some browsers or OS security settings may flag it. You can safely "Allow" or "Run anyway."
